# Our Rock Paper Scissors Game 
A simple and fun Rock Paper Scissors game built with Python and Tkinter!

## Description
This is a classic Rock Paper Scissors game where the user plays against the computer. The game keeps track of your score and displays the results of each round with colorful feedback.

## Features
- Clean and simple GUI (Graphical User Interface)
- Score tracking for both player and computer
- Visual feedback with color-coded results (Green for win, Red for loss, Blue for tie)
- Random computer choices for fair gameplay

## Requirements
- Python 3.x
- tkinter (usually comes pre-installed with Python)

## How to Run
1. Make sure you have Python installed on your computer
2. Download or clone this repository
3. Run the game using:

   python rock_paper_scissors.py

   (Replace `rock_paper_scissors.py` with your actual filename)

## How to Play

1. Click on one of the three buttons: Stone , Paper , or Scissors 
2. The computer will make its choice randomly
3. The result will be displayed on the screen
4. Your scores will be updated automatically
5. Keep playing to see who wins more rounds!

## Game Rules
- Stone beats Scissors
- Scissors beats Paper
- Paper beats Stone
